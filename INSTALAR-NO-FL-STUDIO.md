# Instalação no FL Studio — VST2

O VST2 é um arquivo `.dll`, diferente do pacote em pasta usado pelo VST3.

Estrutura recomendada:

```text
C:\VSTPlugins\Aurora Uplift\Aurora Uplift.dll
```

No FL Studio:

1. `Options > Manage plugins`
2. Adicione `C:\VSTPlugins` em `Plugin search paths`
3. Marque `Verify plugins`
4. Marque `Rescan previously verified plugins`
5. Marque `Rescan plugins with errors`
6. Clique em `Find installed plugins`

A DLL foi preparada para Windows x64. Use o FL Studio de 64 bits.
